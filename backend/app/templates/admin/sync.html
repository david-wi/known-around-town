Sync all salons

Looks up every published salon on Google, matches by name and city, and stores the star rating and review count. Existing ratings are refreshed. Takes 30–90 seconds for the full directory.

Cost: Each salon without a Google Place ID costs 1 Places Text Search call (≈ $0.017). Salons with an existing Place ID cost 1 Place Details call (cheaper). Total cost for {{ total }} salons is under $2.

{# ── How it works ── #}

What syncing does

  • → Searches Google for each salon by name and city
  • → Stores the star rating (e.g. 4.7 ★) and review count in the salon's database record
  • → Ratings appear immediately on salon pages — no restart needed
  • → Google search results will start showing star snippets for salon pages (usually within a week of Google's next crawl)

Run this sync weekly, or after adding new salons to the directory. Ratings don't update automatically.
